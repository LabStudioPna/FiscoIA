1. ¿Qué es FiscoIA?

FiscoIA es una herramienta de análisis contable con inteligencia artificial, pensada para estudios contables, profesionales independientes y pequeños negocios en Argentina. Te ayuda a:

  • Analizar facturas, recibos y documentos contables con IA (extrae CUIT, montos, impuestos automáticamente).
  • Calcular y seguir obligaciones impositivas: IVA, Ganancias, Ingresos Brutos (IIBB), Monotributo, Retenciones.
  • Verificar CAE/QR contra ARCA (Administración Federal de Ingresos Públicos).
  • Generar reportes consolidados por cliente, por período y análisis de tendencias.
  • Descargar datos en Excel, PDF y archivos compatibles con AFIP.

FiscoIA no reemplaza a un contador profesional. Es una herramienta de soporte. La firma, la responsabilidad y las decisiones técnicas siguen siendo del profesional.

2. Arquitectura: tus datos siempre en tu control

FiscoIA no tiene backend propio. No almacenamos tus documentos, tu API key, ni tus análisis en un servidor nuestro. Aquí te explicamos cómo funciona:

Flujo de datos

  • Tu navegador → tu documento (PDF, Excel, imagen) + tu API key de Anthropic → Anthropic (servidores de IA).
  • Anthropic devuelve el análisis (texto, números) a tu navegador.
  • Vos decidís si lo guardás en tu computadora, en Google Drive, o en tu propio Google Sheets.

FiscoIA nunca ve tus documentos originales ni tu API key. Todo ocurre entre tu navegador y Anthropic. Cuando cierras la pestaña, se borra todo.

localStorage y sessionStorage

  • sessionStorage: tu API key se guarda aquí solo mientras la pestaña está abierta. Al cerrar, se borra automáticamente.
  • localStorage: solo guardamos tu preferencia de tema claro/oscuro y tu email de Google. No guardamos análisis ni documentos.

3. Privacidad de tus documentos

¿Dónde van tus facturas y documentos?

Cuando subes una factura a FiscoIA, enviamos el archivo a Anthropic para que su IA lo analice. Anthropic mantiene logs de tus consultas según sus términos de privacidad. Recomendamos revisar esa política.

Sensibilidad de los datos

Las facturas pueden contener información sensible: CUITs, números de transacciones, montos, datos de clientes. Es responsabilidad tuya determinar si esa información puede enviarse a Anthropic. Si tus clientes requieren que sus datos no se procesen fuera de Argentina, el flujo de FiscoIA no es apropiado para esos casos.

Sincronización con Google Sheets

Si habilitas la sincronización automática con Google Sheets, FiscoIA escribe en una hoja que VOS creás y controlás. Los datos viven en tu Google Drive, no en nuestros servidores. Vos podés ver quién tiene acceso.

4. Tu API key de Anthropic

¿Por qué necesitamos tu API key?

FiscoIA no tiene una API propia: usamos la de Anthropic. Tu API key autoriza las consultas de IA y vos pagás directamente a Anthropic por el uso (típicamente centavos por consulta).

¿Dónde va tu API key?

  • Se guarda en sessionStorage de tu navegador.
  • Se envía solo a Anthropic cuando haces una consulta (para autorizar esa consulta).
  • FiscoIA nunca ve tu API key: va directo de tu navegador a Anthropic.
  • Al cerrar la pestaña, se borra de sessionStorage.

Seguridad

  • Usa siempre HTTPS. FiscoIA solo funciona en conexiones seguras.
  • No compartas tu API key con otros. Una persona con tu key puede hacer consultas y que se te facturen.
  • Si creés que tu key fue comprometida, generá una nueva en console.anthropic.com y descartá la antigua.

5. Responsabilidad profesional

FiscoIA es una herramienta de soporte, no asesoramiento fiscal

FiscoIA analiza facturas y tira números. Pero los números no son verdad legal. La IA puede errar, malinterpretar QR, o leer mal un PDF escaneado.

Responsabilidades tuyas

  • Verificar siempre los análisis antes de usarlos en reportes, declaraciones o decisiones.
  • Mantener copias de tus documentos originales.
  • Firmar y sellar los reportes como profesional, con tu matriculación vigente.
  • Cumplir con AFIP, ARCA, provinciales: si FiscoIA dice una cosa y AFIP dice otra, AFIP tiene razón.
  • Registrar auditoria: sabé qué análisis usaste, en qué fecha, y quién los aprobó.

Responsabilidades de FiscoIA

  • Mantener la información técnica actualizada (normas, alícuotas, vencimientos).
  • No almacenar tus datos sin consentimiento.
  • No compartir tu API key ni tus análisis con terceros.

7. Uso aceptable

Qué podés hacer

  • Analizar tus propias facturas y documentos contables.
  • Usar FiscoIA para un estudio contable, empresa o negocio que controles.
  • Generar reportes para clientes auditados por vos.
  • Guardar, descargar y compartir los análisis que generaste.

Qué no podés hacer

  • Compartir tu API key o credenciales de acceso.
  • Usar FiscoIA para procesar datos de terceros sin su consentimiento.
  • Intentar modificar, hackear o romper FiscoIA (ataques, fuerza bruta, etc.).
  • Usar FiscoIA para enviar spam, malware o contenido ilegal.
  • Revender acceso o usar FiscoIA como un servicio propio sin autorización.

8. Cambios en estos términos

Podemos actualizar estos términos en cualquier momento. Los cambios son vinculantes a partir de que se publiquen en esta página. Si usás FiscoIA después del cambio, aceptás los nuevos términos.

Los cambios importantes (tarifas, privacidad, responsabilidad) serán comunicados también por email a tu cuenta.
